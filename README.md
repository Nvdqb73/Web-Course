#**Website Course**

## Using Technology

### **Frontend**

- ReactJs
- Bootstrap

### Backend

- Asp.net core API
- SQL Server

## Description 

The site will primarily focus on showcasing the courses, allowing users to search for a specific course and view the content of the lessons.
